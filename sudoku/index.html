▸ ABOUT SUDOKU

Sudoku — Puzzle Game

Fill the 9×9 grid so every row, column, and 3×3 box contains 1-9 exactly once. Daily puzzle is the same for everyone.

How to play

  • Tap a cell, tap a number to place
  • Hold/long-press a number for a pencil mark
  • Undo via the toolbar

Tips

  • Find a row, column, or box with eight known cells — that's a forced fill
  • Use pencil marks to track candidates per cell
  • When stuck, scan for hidden singles
