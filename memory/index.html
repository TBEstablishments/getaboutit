▸ ABOUT MEMORY

Memory — Memory Game

A face-down grid of pairs. Flip two — match, they stay up. Mismatch, they flip back. Find all pairs in the fewest flips.

How to play

  • Click or tap a card to flip
  • Two cards per turn
  • Game ends when all pairs are found

Tips

  • Walk the grid systematically the first time
  • Map mismatches as you go
  • Larger grids reward methodical scanning over speed
