/* SITE */
GETABOUTIT — a free retro-vaporwave arcade. 16 classic games.
No tracking. No accounts. No build step. No service worker.
Last update: 2026
Standards: HTML5, CSS3, ES2020, JSON-LD schema, OpenGraph, WebManifest
Hosted: Vercel

/* TECH */
Vanilla ES6+ across ~1700 LOC of shared core.
Canvas 2D for every game render path.
Procedural audio via Web Audio API — no asset files.
LocalStorage for scores, streaks, achievements, pinned games.
One typeface: Press Start 2P (Google Fonts).
Ten-color palette: pink, magenta, purple, deep purple, blue, cyan, teal, yellow, orange, red.
